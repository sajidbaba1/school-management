# 🏫 School Management System

A full-stack School Management System built with **Next.js** (Frontend) and **Spring Boot** (Backend).

---

## 🛠️ Tech Stack

| Layer | Technology |
|---|---|
| Frontend Framework | Next.js 14 (App Router) + Tailwind CSS + Shadcn/UI |
| Frontend State | Redux Toolkit |
| Backend Framework | Spring Boot 3 + Spring Security + JWT |
| Database | Neon (PostgreSQL — Serverless) |
| ORM | Spring Data JPA + Hibernate |
| Auth | JWT (JSON Web Tokens) + Role-Based Access |

---

## 📁 Project Structure

```
school-management/
├── frontend/          → Next.js App
├── backend/           → Spring Boot App
└── README.md
```

---

## 👥 Roles

- **Admin** — Manage everything (students, teachers, fees, timetable)
- **Teacher** — Manage marks, attendance, timetable
- **Student** — View results, fees, attendance
- **Parent** — View child's progress

---

## 🚀 Getting Started

### Prerequisites
- Node.js 20+
- Java 17+
- MySQL 8+
- Maven 3.8+

### Run Frontend
```bash
cd frontend
npm install
npm run dev
```

### Run Backend
```bash
cd backend
mvn spring-boot:run
```
